# fcm-introduce-android
본 리포지터리는 블로그 내용을 쉽게 실습하실 수 있도록 안드로이드 환경에서의 FCM 알림 기능 구현 코드를 담은 리포지터리입니다.
## 주의 사항
샘플 코드에는 google-services.json 파일을 저장해두지 않았습니다.  
블로그 글을 참고하여 해당 파일을 다운로드 받은 뒤 app 패키지에 보관해주시기 바랍니다.
